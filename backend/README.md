# Backend

This folder contains the backend part of the project, implemented using **Node.js** and **Express**.

## Project context
- The project was originally designed as a **team project for 6 people**
- The final implementation was completed by **4 team members**
- **The backend was implemented entirely by me**
- AI tools were used during development

## Tech stack
- Node.js
- Express
- Supabase
- dotenv
- CORS
